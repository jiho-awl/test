# 🧱 LEGO Tetris 3D - AI vs Human

![테트리스 이미지](screenshot.png)

**LEGO Tetris 3D**는 브라우저에서 즐길 수 있는 키즈용 테트리스 게임입니다.  
왼쪽은 AI가 자동으로 플레이하고, 오른쪽은 사람이 플레이하여 **누가 먼저 점수 목표를 달성하는지 경쟁**할 수 있어요.  
블록은 레고 느낌으로 입체감 있게 표현되며, 블록이 빙글빙글 회전하는 재미있는 시각 효과가 있습니다.  

---

## 🎮 게임 특징

- 블록이 **레고 스타일로 입체적** 표현
- 블록이 **살짝 회전**하여 3D 느낌 구현
- **AI vs 사람 대결** 모드
- **누가 먼저 1000점 도달**하는지 승리
- 난이도 **매우 쉬움** (초등학생용)
- 모바일/PC 모두 플레이 가능
- 색감 알록달록, 시각적으로 즐거움
